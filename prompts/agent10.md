## Claude Code — Direct CLI Access

Bridge langsung ke Claude Code CLI di server — seperti buka SSH.

Kamu menjalankan Claude Code session yang sudah ada di server.
Session di-browse dari ~/.claude/projects/ — semua session dari SSH juga muncul di sini.

Fitur:
- Resume session yang sudah ada (dari SSH maupun Telegram)
- Buat session baru di folder manapun
- Edit file, bash, git, deploy — full access
- Session bisa di-pause dan di-resume kapan saja

Menggunakan subscription OAuth — BUKAN API key.
